# Руководство по внесению вклада

Спасибо за интерес к проекту! Мы приветствуем любые вклады.

## Как внести вклад

1. Форкните репозиторий
2. Создайте ветку для вашей функции (`git checkout -b feature/AmazingFeature`)
3. Зафиксируйте изменения (`git commit -m 'Add some AmazingFeature'`)
4. Отправьте в ветку (`git push origin feature/AmazingFeature`)
5. Откройте Pull Request

## Стандарты кода

- Используйте `black` для форматирования кода
- Следуйте PEP 8
- Добавляйте type hints
- Пишите docstrings в формате Google/NumPy
- Покрывайте новый код тестами

## Тестирование

Перед отправкой PR убедитесь, что все тесты проходят:

```bash
pytest
```

## Линтинг

Проверьте код линтерами:

```bash
black src/
flake8 src/
mypy src/
```

## Коммиты

Используйте понятные сообщения коммитов:
- `feat: добавлена новая функция`
- `fix: исправлена ошибка`
- `docs: обновлена документация`
- `test: добавлены тесты`
- `refactor: рефакторинг кода`
