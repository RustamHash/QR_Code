# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [2.0.0] - 2024-01-XX

### Добавлено
- Полная реструктуризация проекта с модульной архитектурой
- Обработка текстовых сообщений (одна строка или несколько)
- История обработки файлов и текста
- Статистика для администратора
- Rate limiting для защиты от злоупотреблений
- Валидация входных данных
- Конфигурация через `.env` файл с использованием pydantic-settings
- Структурированное логирование
- Docker контейнеризация
- CI/CD pipeline с GitHub Actions
- Миграции базы данных через Alembic
- Репозитории для работы с данными
- Кастомные исключения
- Type hints во всем коде
- Документация в формате Google/NumPy

### Изменено
- Переход с файлов конфигурации (token.txt, admin_id.txt) на `.env`
- Улучшена архитектура кода (разделение на сервисы, handlers, repositories)
- Улучшена обработка ошибок
- Оптимизирована работа с базой данных

### Улучшено
- Производительность обработки файлов
- Безопасность (валидация файлов, rate limiting)
- UX (лучшие сообщения об ошибках, прогресс обработки)

## [1.0.0] - 2023-XX-XX

### Добавлено
- Базовая функциональность генерации QR-кодов из Excel файлов
- Настройки PDF (ширина, высота, количество QR-кодов)
- Сохранение настроек пользователей
- Интерактивное меню настроек
